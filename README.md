# VISTA Lab website

Source for the VISTA Lab (Visual Intelligence Systems for Trustworthy AI) site,
directed by Dr. Hossein Kashiani at Utah State University.

Live at: https://kashiani.github.io/vista-lab/

## Structure
- `index.html` — single-file site (home carousel, News, People, Research, Join Us tabs)
- `Doc/` — images and logo used by the site

## Editing
- Carousel slides, news items, and people cards are plain HTML blocks inside `index.html` —
  copy/edit/delete the marked sections (see comments in the file).
- To change the logo, replace `Doc/LOGO.png`.

## Deploying
Any push to `main` is served automatically by GitHub Pages
(Settings → Pages → Deploy from branch → `main` / `root`).
